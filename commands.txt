Local Development Commands
1. Start the Backend (Flask API)

    cd backend
    source venv/bin/activate  # Activate virtual environment
    python app.py            # Runs on http://localhost:5000

 - Keep this running in one terminal—it serves the API and connects to the remote DB.

2. Start the Frontend (React PWA)

    cd frontend
    npm run build            # Builds to dist/ with PWA features
    npx serve dist           # Serve locally (install serve: npm i -g serve)

 - Keep this running in another terminal—it hot-reloads on changes.
 - Open http://localhost:5173 in your browser.

3. Build for Production (Optional)

    cd frontend
    npm run build            # Builds to dist/ with PWA features
    npx serve dist           # Serve locally (install serve: npm i -g serve)

4. Test Offline Mode

 - With both running, log in and load inventory (syncs data).
 - Stop the backend (Ctrl+C), refresh the frontend—the app loads offline with cached data.
